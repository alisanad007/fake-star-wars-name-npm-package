## @ali-sanad/fake-star-wars-name

Generate a Fake Star Wars character's Name

### How to use

```
npm install @ali-sanad/fake-star-wars-name
```

## OR

```
yarn add @ali-sanad/fake-star-wars-name
```

```
const { fakeStarWarsName } = require('@ali-sanad/fake-star-wars-name')


fakeStarWarsName(nameParts = 2) => "Leia Jedi"

fakeStarWarsName(nameParts = 5) => "Willard Trooper Dodonna Kabe Brea"
```
